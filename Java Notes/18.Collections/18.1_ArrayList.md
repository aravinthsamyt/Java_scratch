# Java Collections Framework (JCF)

Collections Framework is a set of classes and interfaces used to store and manipulate groups of objects.

Instead of:

```java
int[] arr = new int[100];
```

We use:

```java
ArrayList<Integer> list =
    new ArrayList<>();
```

because collections are dynamic and provide many built-in methods.

---

# Collection Hierarchy

```text
Iterable
   |
Collection
   |
--------------------------------
|              |              |
List          Set           Queue
|              |              |
ArrayList     HashSet      PriorityQueue
LinkedList    LinkedHashSet
Vector        TreeSet
```

Also:

```text
Map
 |
HashMap
LinkedHashMap
TreeMap
Hashtable
```

Map is NOT part of Collection interface.

# ArrayList

Most commonly used collection.

Internally:

```text
Resizable Dynamic Array
```

---

## Why ArrayList?

Array:

```java
int arr[] = new int[5];
```

Fixed size.

ArrayList:

```java
ArrayList<Integer> list =
    new ArrayList<>();
```

Dynamic size.

---

# Creating ArrayList

```java
import java.util.ArrayList;

ArrayList<Integer> list =
    new ArrayList<>();
```

String example:

```java
ArrayList<String> names =
    new ArrayList<>();
```

# add()

Adds element at end.

```java
list.add(10);
list.add(20);
list.add(30);

System.out.println(list);
```

Output:

```text
[10, 20, 30]
```

---

# add(index, element)

```java
list.add(1, 100);

System.out.println(list);
```

Output:

```text
[10, 100, 20, 30]
```

---

# get()

Access element.

```java
System.out.println(
    list.get(2)
);
```

Output:

```text

```

---

# set()

```java

System.out.println(list);
```


---

# remove(index)



# remove(Object)

```java
list.remove(Integer.valueOf(20));
```

Output:

```text
[10, 30]
```

Important:

```java
list.remove(1);
```

removes index 1.

```java
list.remove(Integer.valueOf(1));
```

removes value 1.

# Iterating ArrayList

## For Loop

```java
for(int i=0;i<list.size();i++){

    System.out.println(
        list.get(i)
    );
}
```

---

## Enhanced For Loop

```java
for(int x : list){

    System.out.println(x);
}
```

---

## Iterator

```java
Iterator<Integer> it =
list.iterator();

while(it.hasNext()){

    System.out.println(
        it.next()
    );
}
```

# Time Complexity

| Operation     | Complexity |
| ------------- | ---------- |
| get(i)        | O(1)       |
| set(i)        | O(1)       |
| add(end)      | O(1) avg   |
| add(index)    | O(n)       |
| remove(index) | O(n)       |
| contains()    | O(n)       |

---

# Array vs ArrayList

| Feature           | Array | ArrayList       |
| ----------------- | ----- | --------------- |
| Size              | Fixed | Dynamic         |
| Methods           | Few   | Many            |
| Primitive Support | Yes   | Wrapper Classes |
| Flexibility       | Low   | High            |
